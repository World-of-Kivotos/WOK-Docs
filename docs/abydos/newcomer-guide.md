# 阿拜多斯高等学院新人引导

::: tip 欢迎加入阿拜多斯高等学院！
本指南将帮助你快速了解学院位置、生活须知与重要规则。
:::

## 🧭 前往学院

### 步骤 1：打开传送菜单

在游戏中输入以下指令：

```bash
/wok warp
```

### 步骤 2：传送到阿拜多斯东北矿山

在传送菜单中选择传送至 **阿拜多斯东北矿山**。

### 步骤 3：前往学院

1. 沿主干道向 **西南方向** 行走
2. 在第一个铺装路面的岔路口向内走即可到达

::: warning 安全提示
夜晚路面有怪物出没，请注意安全！⚔️
:::

## 🏠 到达学院与设置重生点

到达自管区后，请立即设置重生点：

### 设置家的位置

```bash
/sethome
```

### 传送回家

```bash
/home
```

::: info 重要提醒
只能设置一个传送点，新设置的点会覆盖旧点。📍
:::

## 🚫 禁止行为

::: danger 严格禁止
以下行为在自管区内严格禁止，违反者将受到处罚：
:::

- ❌ **饮用"苦力怕的威压"**（绿色瓶装饮料，别名"空爆"）
  - 会严重破坏地形，造成不可逆的损害
- ❌ **私自使用爆炸物**
  - 需向学院管理层报备批准后方可使用
- ❌ **禁止一切生电活动**
  - 这是强制性规定，请务必遵守
- ❌ **建造或使用机械动力相关机器**
  - 仅允许使用装饰性方块
- ❌ **使用矿物透视**
  - 任何形式的透视类作弊均被禁止

## 🛠️ 其他实用信息

### 1. 新手礼包 🎁

打开物品栏，点击左上角 📖 图标领取服务器新手礼包。

### 2. 建材市场 🏪

前往 **DU建材市场**（可传送），绝大多数建材价格低廉，适合新人采购。

### 3. 武器装备 🔫

- DU及各大学院均有枪店
- 部分学院（如圣三一、千年）有特殊NPC可兑换强力装备

### 4. 获取帮助 💬

遇到问题可在群内提问，管理员与其他成员都很热心，会及时为你解答。
