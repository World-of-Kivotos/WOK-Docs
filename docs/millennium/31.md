# 圣三一综合学园新人引导

::: tip 欢迎加入圣三一综合学园！
以下是你现在所必做的入学准备，请仔细阅读。
:::

## 🔐 账号注册

首次进入服务器，请先注册账号：

```bash
/register [密码] [重复密码]
```

注册完成后，下次登录只需输入：

```bash
/login [密码]
```

## 🏛️ 前往学院自管区

> 自管区是大家生活的地方，你可以在这里进行生存、建造等活动。

学院当前有**新区**和**老区**两块，距离 DU（出生点）最近的是老区。

### 前往方式

1. **领取新手礼包**  
   打开物品栏，在屏幕左上角找到 📖 图标（任务书），里面有新手礼包可以领取
   - 包含食物和水，可以支持你跑到自管区
   - 如果不够，可以击杀路边的动物获取食物

2. **选择交通方式**  
   自管区位于坐标：`2698, 70, -3404`
   - **步行**：自己步行前往或找人带路
   - **地铁**：DU 的地铁站（白鸟西站）位于初始出生点的西边，请看清站点，不要坐错

3. **安全提示**  
   ::: warning 道路千万条，安全第一条
   注意旅途中的怪物和环境危险
   :::

## 🏠 设置重生点

到达自管区后，请立即设置重生点：

```bash
/sethome
```

::: info 重生点说明
- 该指令可以**无限次使用**
- 但最多只能设置**一个**重生点
- 新输入的指令会**覆盖**上一次设置的重生点
:::

需要回到重生点时，输入：

```bash
/home
```

## ⚠️ 注意事项

::: danger 严格禁止的行为
以下行为严格禁止，违者将受到处罚：

- **禁止在建筑附近喝"苦力怕的威压"**（别名：空爆精酿）  
  外形为绿色瓶装饮料，会严重破坏地形及损害他人财产  
  如确需使用，必须上报管理组

- **严禁使用机械动力**  
  可以使用机械动力的装饰方块进行建筑，但不可以打造机器

- **严禁使用任何生电机器**  
  任何自动化的建筑都必须严格控制规模

- **严禁使用矿透**  
  缺钱可以直接向其他玩家或管理组寻求帮助
:::
