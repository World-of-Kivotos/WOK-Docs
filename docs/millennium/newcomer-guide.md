# 千年科技学院新人引导

::: tip 欢迎来到千年科技学院！
本指南将帮助新人快速了解如何前往千年科技学院并开始你的学院生活。
:::

## 🚀 前往千年科技学院

### 步骤 1：打开传送菜单

在游戏中输入以下指令：

```bash
/wok warp
```

这将打开传送菜单，选择前往 **阿拜多斯东北矿山**。

### 步骤 2：前往阿拜多斯

传送到阿拜多斯东北矿山后，按照以下路线前进：

1. 向 **西南方向** 前进
2. 沿着马路走，目标是 **阿拜多斯**
3. 找到阿拜多斯的车站

::: info 导航提示
沿着主干道前进，很容易就能找到阿拜多斯车站
:::

### 步骤 3：乘车前往千年学院

到达阿拜多斯车站后：

1. 在阿拜多斯车站乘车
2. 选择路线：**阿拜多斯 → 千年学院**
3. 等待列车到达千年学院

### 步骤 4：转乘到千年自管区

在千年学院站：

1. 下车后前往站台
2. 选择路线：**千年学院 → 千年自管区**
3. 乘车前往千年自管区

::: tip 千年小卖部
在千年学院转乘时，如果上一层楼，你会发现**千年小卖部**，那里有卖枪的商店！记得去看看~ 🔫
:::

---

## 🏠 到达千年自管区

恭喜！你已经成功到达千年科技学院的自管区域。

### 设置重生点

到达自管区后，建议立即设置你的家：

```bash
# 设置家的位置
/sethome
```

需要回家时，输入：

```bash
# 传送回家
/home
```

::: info 重生点说明
- 该指令可以**无限次使用**
- 但最多只能设置**一个**重生点
- 新输入的指令会**覆盖**上一次设置的重生点
:::
